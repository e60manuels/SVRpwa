# PWA Installatie Prompt - Direct Tonen bij Eerste Bezoek

## Het Probleem met Browser Heuristics

De standaard `beforeinstallprompt` event wordt door browsers gecontroleerd via "engagement heuristics":

### Chrome/Edge Criteria:
- **30 seconden** minimaal gebruikers-engagement
- Site moet bezocht zijn op **minimaal 2 verschillende dagen** (in sommige versies)
- Gebruiker moet de site **meerdere keren** hebben bezocht
- Browser bepaalt zelf wanneer het event wordt getriggerd

### Het Probleem:
Je kunt **niet forceren** wanneer `beforeinstallprompt` getriggerd wordt. Dit betekent dat je banner misschien niet verschijnt bij het eerste bezoek.

## De Oplossing: Hybrid Approach

De aangepaste implementatie combineert twee strategieën:

### 1. Toon Banner Onmiddellijk
```javascript
document.addEventListener('DOMContentLoaded', () => {
    if (!isAppInstalled()) {
        // TOON DE BANNER DIRECT - wacht niet op beforeinstallprompt
        showInstallBanner();
    }
});
```

### 2. Gebruik Native Prompt Wanneer Beschikbaar
```javascript
window.addEventListener('beforeinstallprompt', (e) => {
    e.preventDefault();
    deferredPrompt = e; // Sla op voor later gebruik
    // MAAR toon de banner NIET hier - die is al getoond
});
```

### 3. Fallback voor Alle Browsers
```javascript
async function triggerInstall() {
    if (deferredPrompt) {
        // Native prompt beschikbaar - gebruik die
        deferredPrompt.prompt();
    } else {
        // Geen native prompt - toon browser-specifieke instructies
        showBrowserSpecificInstructions();
    }
}
```

## Hoe Het Werkt

### Scenario 1: Chrome/Edge (Native Support)
```
1. Pagina laadt → Banner verschijnt DIRECT
2. Gebruiker klikt "Installeren"
3. Als beforeinstallprompt beschikbaar → Native dialog
4. Anders → Toon instructies (menu → installeren)
```

### Scenario 2: iOS Safari
```
1. Pagina laadt → Banner verschijnt DIRECT
2. Gebruiker klikt "Installeren"
3. Toon iOS-specifieke instructies (Share → Add to Home Screen)
```

### Scenario 3: Andere Browsers
```
1. Pagina laadt → Banner verschijnt DIRECT
2. Gebruiker klikt "Installeren"
3. Toon browser-specifieke instructies of bookmark optie
```

## Voordelen van Deze Aanpak

### ✅ Directe Zichtbaarheid
- Banner verschijnt **onmiddellijk** bij eerste bezoek
- Geen wachten op browser heuristics
- Maximale conversie kans

### ✅ Beste van Beide Werelden
- Gebruikt native prompt wanneer beschikbaar
- Fallback naar instructies wanneer niet beschikbaar

## Code Overzicht

### Banner Direct Tonen
```javascript
function showInstallBanner() {
    if (isAppInstalled()) return;
    if (!shouldShowBanner()) return;
    
    // Toon banner ONMIDDELLIJK - niet wachten op events
    installBanner.classList.add('show');
    trackInstallPromptShown();
}
```

### Smart Install Trigger
```javascript
async function triggerInstall() {
    // iOS heeft speciale behandeling nodig
    if (isIOS() && !isAppInstalled()) {
        showIOSInstructions();
        return;
    }
    
    // Gebruik native prompt als beschikbaar
    if (deferredPrompt) {
        const { outcome } = await deferredPrompt.prompt();
        // Handle result...
    } else {
        // Geen native support - toon instructies
        showBrowserSpecificInstructions();
    }
}
```

### Browser-Specifieke Instructies
```javascript
function showBrowserSpecificInstructions() {
    const browserName = getBrowserInfo();
    let instructions = '';
    
    if (isChromium()) {
        instructions = `
            1. Klik op menu (⋮)
            2. Selecteer "Installeren..."
            3. Bevestig installatie
        `;
    } else if (isSafari()) {
        instructions = `
            1. Klik op deel-icoon
            2. Selecteer "Voeg toe aan Dock/Beginscherm"
            3. Bevestig
        `;
    }
    
    // Toon in modal
    showInstructionsModal(instructions);
}
```

## LocalStorage Management

### Herinner Voorkeur
```javascript
function shouldShowBanner() {
    const dismissed = localStorage.getItem('install-banner-dismissed');
    const dismissedDate = localStorage.getItem('install-banner-dismissed-date');
    
    if (dismissed && dismissedDate) {
        // Toon opnieuw na 90 dagen
        const daysSinceDismissed = (Date.now() - parseInt(dismissedDate)) / (1000 * 60 * 60 * 24);
        return daysSinceDismissed > 90;
    }
    
    return true; // Eerste keer - toon altijd
}
```

### Reset Optie
```javascript
// Voor development/testing
function resetInstallPrompt() {
    localStorage.removeItem('install-banner-dismissed');
    localStorage.removeItem('install-banner-dismissed-date');
    window.location.reload();
}

// Roep aan in console: resetInstallPrompt()
```

## Platform Detection

### Detectie Functies
```javascript
function isIOS() {
    return /iPad|iPhone|iPod/.test(navigator.userAgent) && !window.MSStream;
}

function isChromium() {
    return /Chrome/.test(navigator.userAgent) && /Google Inc/.test(navigator.vendor);
}

function isSafari() {
    return /^((?!chrome|android).)*safari/i.test(navigator.userAgent);
}

function isAppInstalled() {
    // Standalone mode check
    if (window.matchMedia('(display-mode: standalone)').matches) {
        return true;
    }
    // iOS standalone
    if (window.navigator.standalone === true) {
        return true;
    }
    return false;
}
```

## Veelvoorkomende Problemen & Oplossingen

### "Banner verschijnt niet"
**Mogelijke oorzaken:**
1. App is al geïnstalleerd → Check `isAppInstalled()`
2. Banner is recent gesloten → Check localStorage
3. CSS display property → Check `.show` class

**Debug:**
```javascript
console.log('Is installed:', isAppInstalled());
console.log('Should show:', shouldShowBanner());
console.log('Banner element:', installBanner);
```

### "Install button doet niets op Chrome"
**Mogelijke oorzaken:**
1. Service Worker niet geregistreerd
2. Manifest niet geldig
3. Niet op HTTPS

**Debug:**
```javascript
// Check service worker
navigator.serviceWorker.getRegistrations().then(r => console.log(r));

// Check manifest
fetch('/manifest.json').then(r => r.json()).then(console.log);
```

### "iOS instructies verschijnen niet"
**Check iOS detectie:**
```javascript
console.log('Is iOS:', isIOS());
console.log('User Agent:', navigator.userAgent);
```

## Advanced: Progressive Disclosure

Toon banner alleen na waarde-demonstratie:

```javascript
// Track belangrijke acties
let valueDemonstrated = false;

function onImportantAction() {
    // Bijvoorbeeld: na eerste succesvolle taak
    valueDemonstrated = true;
    localStorage.setItem('value-demonstrated', 'true');
    
    // Nu pas banner tonen

        showInstallBanner();
    }

// Bij page load
```
